## Description
### The Task Manager is a Python program that allows registering users, adding tasks, and generating reports. It allows users to register, assign tasks, view tasks, and perform various operations on them.

##
##

## Features

### User registration: Users can register with a unique username and password.
### Task management: Users can add tasks, view all tasks, view their assigned tasks, and mark tasks as complete.
### Reports: Users can generate reports on task statistics and user overviews.

## Usage

## 1. Run the program:
#### shell
### python task_manager.py

##
##

## Upon running the program, you will see the following menu:
## Select an option by entering the corresponding letter and press Enter.
##
## 2. Follow the on-screen instructions to perform various operations:
###   a. Register a user: Select option 'r' to register a new user.
###   b. Add a task: Select option 'a' to add a new task.
###   c. View all tasks: Select option 'va' to view all tasks.
###   d. View assigned tasks: Select option 'vm' to view your assigned tasks.
###   e. Generate reports: Select option 'gr' to generate reports.
###   f. Display statistics: Select option 'ds' to display statistics.
###   g. Exit: Select option 'e' to exit the program.

##

##  a. If you choose to register a user ('r'), you will be prompted to enter a username and password. Ensure that the username is unique.

##

## b. If you choose to add a task ('a'), you will be prompted to enter the username of the person the task is assigned to, along with other task details.

##

## c. If you choose to view all tasks ('va'), you will see a list of all the tasks stored in the 'tasks.txt' file.

##

## d. If you choose to view your assigned tasks ('vm'), you will be prompted to enter your username and see a list of tasks assigned to you.

##

## e. If you choose to generate reports ('gr'), two text files will be generated:
'task_overview.txt' and 'user_overview.txt'. These files contain relevant information and statistics about tasks and users.

##

## f. If you choose to display statistics ('ds'), the program will display task and user statistics on the console, including the total number of tasks, completed tasks, overdue tasks, etc. 

##

## g. Select the exit option to exit the program ('e'), and the program will terminate.
